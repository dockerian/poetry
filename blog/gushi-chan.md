# 禅诗三百首 <a name="title"></a>
<!--Recommend to use Markdown viewer or plugin in browser-->

> 汇集历代禅宗高僧与文人悟道、寄情的经典诗作，以空灵旷达的意境著称，常借助山水、自然来阐释禅理。其中最为大众熟知的名篇如：神秀大师《[示法偈](#shifajie)》、六祖惠能大师的《[无相偈](#wuxiang)》、布袋和尚的《[插秧诗](#chayang)》、灵一《[题僧院](tisengyuan)》、李翱《[赠药山高僧惟俨](#zengyaoshan)》、翁卷《[野望](#yewang)》等，深刻影响中国人的处世哲学。原摘：[佛弟子文库](https://fodizi.net/qt/qita/26707.html)；另见知乎《[禅意诗百首讲解](gushi-zen.md)》。

---

### 终南望余雪｜祖咏

```
终南阴岭秀，积雪浮雲端。
林表明霁色，城中增暮寒。
```

### 一花开五叶｜菩提达摩

```
吾本来兹土，传法救迷情。
一花开五叶，结果自然成。
```

### 咏怀诗｜支遁

```
端坐邻孤影，眇罔玄思劬。
偃骞收神辔，领略综名书。
涉老咳双玄，披壮玩太初。
咏发清风集，触思皆恬愉。
俯欣质文蔚，仰悲二匠徂。
萧萧柱下迥，寂寂蒙邑虚。
廓矣千裁事，消液归空无。
无矣复何伤，万殊归一途。
道会贵冥想，罔象掇玄珠。
怅怏浊水际，几忘映清渠。
反鉴归澄漠，容与含道符。
心与理理密，形与物物疏。
萧索人事去，独与神明居。
```

### 庐山东林杂诗｜慧远

```
崇岩吐清气，幽岫栖神迹。
希声奏群籁，响出山溜滴。
有客独冥游，径然思所适，
挥手抚雲门，灵式安足辟。
流心扣玄扃，感至理弗隔。
孰是腾九霄，不奋冲天翮。
妙同趣自均，一悟超三益。
```

### 陵峰采药触兴为诗｜帛道猷

```
连峰数千里，修林带平津。
雲过远山翳，风至梗荒榛。
茅茨隐不见，鸡鸣知有人。
闲步践其径，处处见遗薪。
始知百代下，故有上皇民。
```

### 石壁精舍还湖中作｜谢灵运

```
昏旦变气候，山水含清晖。
清晖能娱人，游子憺忘归。
出谷日尚早，气舟阳已微。
林壑敛暝色，雲霞收夕霏。
芰荷迭映蔚，蒲稗相因依。
披拂趋南径，愉悦偃东扉。
虑澹物自轻，意惬理无违。
寄言摄生客，试用此道推。
```

### 咏孤石｜高丽定法师

```
迥石直生空，平湖四望通。
岩根恒洒浪，树杪镇摇风。
偃流还渍影，浸霞更上红。
独拔群峰外，孤秀白雲中。
```

### 空手把锄头｜善慧

```
空手把锄头，步行骑水牛。
人从桥上走，桥流水不流。
```

### 游少林寺｜沈佺期

```
长歌游宝地，屐倚对珠林。
雁塔霜风古，龙池岁月深。
绀园澄夕霁，碧殿下秋阴。
归路霞光晚，山蝉处处吟。
```

### 山夜闻钟｜张说

```
夜卧闻夜钟，夜静山更响。
霜风吹寒月，窈窕虚中上。
前声既春容，後声复晃荡。
听之如可见，寻之定无像。
信知本际空，徒挂生灭想。
```

### 寒山多幽奇｜寒山

```
寒山多幽奇，登者但恒慑。
月照水澄澄，风吹草猎猎。
凋梅雪作花，杌木雲充叶。
触雨转鲜灵，非晴不可涉。
```

### 终不逐时情｜拾得

```
般若酒泠泠，饮多人易醒。
余住天台山，凡愚哪见形？
常游深谷洞，终不逐时情。
无思亦无虑，无辱也无荣。
```

### 拆却空王殿｜圣可玉

```
拆却空王殿，掀翻狮子床。
太平无忌讳，在在百花香。
```

### 经废宝庆寺｜司空曙

```
黄叶前朝寺，无僧寒殿开。
池晴龟出曝，松暝鹤飞回。
古砌碑横草，阴廊画杂苔。
禅宫亦销歇，尘世转堪哀。
```

### 送普上人还阳羡｜皇甫曾

```
花宫难久别，道者忆千灯。
残雪入林路，暮山归寺僧。
日光依嫩草，泉响滴春冰。
何用求方便，看心是一乘。
```

### 同苗发慈恩寺避暑｜李端

```
追凉寻宝刹，畏日望璇题。
卧草同鸳侣，临池似虎溪。
树闲人迹外，山晚鸟行西。
若问无心法，莲花隔淤泥。
```

### 听嘉陵江水声寄深上人｜韦应物

```
凿崖泄奔湍，古称神禹迹。
夜喧山门店，独宿不安席。
水性自雲静，石中本无声。
如何两相激，雷转空山惊？
贻之道门归，了此物我情。
```

### 天姥岑望天台山｜灵澈

```
天台众峰外，华顶当寒空。
有时半不见，崔嵬在雲中。
```

### 哭柏岩禅师｜李益

```
遍与傍人别，临终尽不愁。
影堂谁为扫，坐塔自看修。
白日钟边晚，青苔钵上秋。
天涯禅弟子，空到柏岩游。
```

### 题破山寺後禅院｜常建

```
清晨入古寺，初日照高林。
竹径通幽处，禅房花木深。
山光悦鸟性，潭影空人心。
万籁此俱寂，但余钟磬音。
```

### 月夜泛舟｜法振

```
西塞长雲尽，南湖片月斜。
漾舟人不见，卧入武陵花。
```

### 赠眼医婆罗门僧｜刘禹锡

```
三秋伤望远，终日泣途穷。
两目今先暗，中年似老翁。
看朱渐成碧，羞日不禁风。
师有金篦术，如何为发蒙？
```

### 送灵一上人｜陈羽

```
十年劳远别，一笑喜相逢。
又上青山去，青山千万重。
```

### 江雪｜柳宗元

```
千山鸟飞绝，万径人踪灭。
孤舟蓑笠翁，独钓寒江雪。
```

### 望牛头寺｜杜甫

```
牛头见鹤林，梯径绕幽林。
春色浮天外，天河宿殿阴。
传灯无白日，布地有黄金。
休作狂歌老，回看不住心。
```

### 宿山寺｜贾岛

```
众岫耸寒色，精庐向此分。
流星透疏木，走月逆行雲。
绝顶人来少，高松鹤不群。
一僧年八十，世事未曾闻。
```

### 闲居｜姚合

```
不自识疏弊，终年住在城。
过门无马迹，满宅是蝉声。
带病吟虽苦，休官梦已清。
何当学禅观，依止古先生？
```

### 新年｜无可

```
燃灯朝复夕，渐作长年身。
紫阁未归日，青门又见春。
掩关寒过尽，开定草生新。
自有林中趣，谁惊岁去频。
```

### 杂诗｜庞蕴

```
日用事无别，唯吾自偶谐。
头头非取舍，处处勿张乖。
朱紫谁为号，青山绝点埃。
神通并妙用，运水与搬柴。
```

### 日本病僧｜项斯

```
雲水绝归路，来时风送船。
不专负後事，犹坐病中禅。
深壁藏灯影，空窗出艾烟。
已无乡土信，起塔寺门前。
```

### 山居诗｜常达

```
身闲依祖寺，志辟性多慵。
少室遗真旨，层楼起暮钟。
啜茶思好水，对月数诸峰。
有问山中趣，庭前是古松。
```

### 击竹悟道偈｜智闲

```
一击忘所知，更不假修持。
动容扬古道，不堕悄然机。
处处无踪迹，声色外威仪。
诸方达道者，咸言上上机。
```

### 示法偈｜神秀大师 <a name="shifajie"></a>

```
身是菩提树，心如明镜台。
时时勤拂拭，莫使有尘埃。
```

### 无相偈｜六祖惠能大师 <a name="wuxiang"></a>

```
菩提本无树，明镜亦非台。
本来无一物，何处惹尘埃。
```

### 壁上诗｜封干

```
本来无一物，亦无尘可拂。
若能了达此，不用坐兀兀。
```

### 僧院花｜白居易

```
欲悟色空为佛事，故栽芳树在僧家。
细看便是华严偈，方便风开智慧花。
```

### 赠衡岳僧｜怀素

```
祝融高座对寒峰，雲水昭丘几万重。
五月衲衣犹近火，起来白鹤冷清松。
```

### 题僧院｜灵一 <a name="tisengyuan"></a>

```
虎溪闲月引相过，带雪松枝挂薜萝。
无限青山行欲尽，白雲深处老僧多。
```

### 简寂观｜灵彻

```
古松古柏岩壁间，猿攀鹤巢古枝折。
五月有霜六月寒，时见山翁来取雪。
```

### 法华寺上方题江上人禅室｜皎然

```
路入松声远更奇，山光水色共参差。
中峰禅寂一僧在，坐对梁朝老挂枝。
```

### 欲归山留别韩潮州愈偈｜大颠

```
辞君莫怪归山早，为忆松萝对月宫。
台殿不将金锁闭，来时自有白雲封。
```

### 马上作｜贯休

```
柳岸花堤夕照红，风清襟袖辔璁珑。
行人莫讶频回音，家在凝岚一点中。
```

### 木仙庵吟诗｜吴承恩

```
禅心似月迥无尘，诗兴如天清更新。
半枕松风茶未熟，吟怀潇洒满腔春。
```

### 山中作｜处默

```
席帘高卷枕高攲，门掩垂萝蘸碧溪。
闲把史书眠一觉，起来山日过松西。
```

### 孤猿叫落中岩月偈｜延寿

```
孤猿叫落中岩月，野客吟残半夜灯。
此境此时谁得意？白雲深处坐禅僧。
```

### 千尺丝纶｜德诚

```
千尺丝纶直下垂，一波才动万波随。
夜静水寒鱼不食，满船空载月明归。
```

### 春晚书山家屋壁｜贯休

```
柴门寂寂黍饭馨，山家烟火春雨晴。
庭花蒙蒙水泠泠，小儿啼索树上莺。
```

### 赠质上人｜杜荀鹤

```
枿坐雲游出世尘，兼无瓶钵可随身。
逢人不说人间事，便是人间无事人。
```

### 溪上遇雨二首｜崔道融

```
坐看黑雲衔猛雨，喷洒前山次独晴。
忽惊雲雨在头上，却是山前晚照明。
```

### 溪居即事｜崔道融

```
篱外谁家不系船，春风吹入钓鱼湾。
小童疑是有村客，急向柴门去却关。
```

### 千年石上古人踪｜寒山

```
千年石上古人踪，万丈岩前一点空。
明月照时常皎洁，不老寻讨问西东。
```

### 无去无来｜拾得

```
无去无来本湛然，不居内外及中间；
一颗水精洁暇翳，光明透出满人间。
```

### 沩山牯牛｜百丈怀海

```
放出沩山水牯牛，无人坚执绳鼻头。
绿杨芳草春风岸，高卧横眠得自由。
```

### 鱼鼓颂｜赵州从谂

```
四大由来造化功，有声全贵里头空。
莫嫌不与凡夫说，只为宫商调不同。
```

### 答僧偈｜大梅法常

```
一池荷叶衣无尽，满树松花食有余。
刚被世人知去处，又移茅舍入深居。
```

### 落雪临风｜清江

```
落雪临风不厌看，更多还恐蔽林峦。
愁人正在书窗下，一片飞来一片寒。
```

### 来时无迹去无踪｜鸟巢禅师

```
来时无迹去无踪，去与来时事一同。
何须更问浮生事？只此浮生是梦中。
```

### 无题｜无尽藏比丘尼

```
终日寻春不见春，芒鞋踏破岭头雲；
归来偶把梅花嗅，春在枝头已十分。
```

### 心如广大｜黄檗希运

```
心如大海无边际，广植净莲养身心。
自有一双无事手，为作世间慈悲人。
```

### 辞世偈｜洞山良价

```
学者恒沙无一悟，过在寻他舌头路。
欲得忘形泯踪迹，努力勤殷空里步。
```

### 学道先须且学贫｜龙牙居遁

```
学道先须且学贫，学贫贫後道方亲。
一朝体得成贫道，道用还如贫底人。
```

### 学道｜龙牙居遁

```
夫人学道莫贪求，万事无心道合头。
无心始体无心道，体得无心道亦休。
```

### 精舍遇雨｜可止

```
空门寂寂淡吾身，溪雨微微洗客尘。
卧向白雲情未尽，任他黄鸟醉芳春。
```

### 嗅梅｜无尽藏

```
尽日寻春不见春，芒鞋踏遍陇头雲。
归来笑牛梅花嗅，春在枝头已十分。
```

### 悟禅诗｜龙山禅师

```
三间茅屋从来住，一道神光万境闲。
莫把是非来辨我，浮世穿凿不相关。
```

### 我向前溪照碧流｜寒山

```
我向前溪照碧流，或向岩边坐盘石。
心似孤雲无所依，悠悠世事何须觅。
```

### 不是众生不是相｜大川禅师

```
不是众生不是相，春暖黄莺啼柳上。
说尽山河海月情，依前不会还怅惆。
休惆怅，万里无雲天一样。
```

### 藉婆衫子拜婆年｜大川禅师

```
藉婆衫子拜婆年，礼数周旋已十分。
竹影扫阶尘不动，月轮穿沼水无痕。
```

### 似水如雲一梦身｜大川禅师

```
似水如雲一梦身，不知此外更何亲。
个中不许容他物，分付黄梅路上人。
```

### 春行即兴｜李华

```
宜阳城下草萋萋，涧水东流复向西。
芳树无人花自落，春山一路鸟空啼。
```

### 渔翁｜柳宗元

```
渔翁夜傍西岩宿，晓汲清湘燃楚竹。
烟销日出不见人，欸乃一声山水绿。
回看天际下中流，岩上无心雲相逐。
```

### 秋夜宿破山寺｜皎然

```
秋风落叶满空山，古寺残灯石壁间。
昔日经行人去尽，寒雲夜夜自飞还。
```

### 画松｜景雲

```
画松一似真松树，且待寻思记得无？
曾在天台山上见，石桥南畔第三株。
```

### 再游玄都观｜刘禹锡

```
百亩庭中半是苔，桃花净尽菜花开。
种桃道士归何处？前度刘郎今又来。
```

### 本来照｜香严禅师

```
拟心开口隔山河，寂寞无言也被呵。
舒展无穷又无尽，卷来绝迹已成多。
```

### 滁州西涧｜韦应物

```
独怜幽草涧边生，上有黄鹂深树鸣。
春潮带雨晚来急，野渡无人舟自横。
```

### 过融上人兰若｜孟浩然

```
山头禅室挂僧衣，窗外无人溪鸟飞。
黄昏半在下山路，却听泉声恋翠微。
```

### 山中与幽人对酌｜李白

```
两人对酌山花开，一杯一杯复一杯。
我醉欲眠卿且去，明朝有意抱琴来。
```

### 游洞庭湖五首其二｜李白

```
南湖秋水夜无烟，耐可乘流直上天。
且就洞庭赊月色，将船买酒白雲边。
```

### 与史郎中钦听黄鹤楼上吹笛/黄鹤楼闻笛｜李白

```
一为迁客去长沙，西望长安不见家。
黄鹤楼上吹玉笛，江城五月落梅花。
```

### 问余何意栖碧山｜李白

```
问余何意栖碧山，笑而不答心自闲。
桃花流水窅然去，别有天地非人间。
```

### 晚秋闲居｜白居易

```
地僻门深少送迎，披衣闲坐养幽情。

日月升沉不住，白雲来去常闲。
```

### 翠雲庵｜永觉元贤

```
翠微深处便为家，石涧潺浣洗落霞。
门外有山皆种竹，庵前无隙可栽花。
铁牛踞地休加策，石鼓悬空勿用挝。
且学寒蝉甘自闭，客来不点赵州茶。
```

### 妙高｜南怀瑾

```
妙高峰顶路难寻，万转千回枉用心。
偶傍清溪闲处立，一声啼鸟落花深。
```

### 伯劳西去｜大成

```
伯劳西去雁东来，李白桃红岁岁开。
万事无过随分好，人生何用苦安排。
```

### 江村即事｜司空曙

```
钓罢归来不系船，江村月落正堪眠。
纵然一夜风吹去，只在芦花浅水边。
```

### 闲田地｜法演

```
山前一片闲田地，叉手叮咛问祖翁。
几度卖来还自买，为怜松竹引清风。
```

### 茅庵｜济水洗

```
茅庵隐隐傍溪边，拨雾寻芝种石田。
转步看花花欲醉，流莺不觉落窗前。
```

### 火筒汝风杲

```
两头截断见心空，一窍能生万灶风。
漫借渠侬伸口气，死柴口上也通红。
```

### 赠别何庵

```
万里波澜一叶舟，秋风如箭月如钩。
不须更问途中事，过得三巴即便休。
```

### 八十｜香幢海禅师

```
八十婆婆未嫁人，柔肠百结向谁伸。
而今趁得东君力，喜见眉间一点春。
```

### 兴至｜谷隐启

```
兴至持竿驾小舟，乘风泛入五湖秋。
橹声惊起芦花雁，却逐流沙达别洲。
```

### 问处｜惠因静

```
问处分明答处亲，半同含笑半同嗔。
君看陌上二三月，那树枝头不带春。
```

### 黄梅抚松

```
黄梅时节家家雨，春草池塘处处蛙。
白鹭下田千点雪，黄鹂上树一枝花。
```

### 送慧上人往上江籴麻米｜宏智正觉

```
芦花明月水茫茫，激箭风舟破冷光。
亲到庐陵酬米价，那时开口便相当。
```

### 和梁山十牛颂｜楚石梵琦

```
前坡只尺是侬家，叠叠春山横暮霞。
好个归来时节子，一钩新月挂檐牙。
```

### 示法诗｜希迁

```
从来共住不知名，任运相将只么行。
自古上贤犹不识，造次凡流岂能明？
```

### 枫桥夜泊｜张继

```
月落乌啼霜满天，江枫渔火对愁眠。
姑苏城外寒山寺，夜半钟声到客船。
```

### 沩山牛｜怀海

```
放出沩山水牯牛，无人坚执鼻绳头。
绿杨芳草春风岸，高卧横眠得自由。
```

### 哭空寂寺玄上人｜钱起

```
凄然双树下，垂泪远公房。
灯续生前火，炉添没後香。
阴阶明片雪，寒竹响空廊。
寂灭应为乐，尘心徒自伤。
```

### 题张僧繇醉僧图｜怀素

```
人人送酒不曾沽，终日松间挂一壶。
草圣欲成狂便发，真堪画入醉僧图。
```

### 项王庙｜灵一

```
缅想咸阳事可嗟，楚歌哀怨思无涯。
八千弟子归何处，万里鸿沟属汉家。
弓断阵前争日月，血流垓下定龙蛇。
拔山力尽乌江水，今日悠悠空浪花。
```

### 几度逢春｜法常

```
摧残枯木倚寒林，几度逢春不变心。
樵客遇之犹不顾，郢人哪得苦追寻？
一池荷叶衣无尽，数树松花食有余。
刚被世人知住处，又移茅舍入深居。
```

### 题鹤林寺僧舍｜李涉

```
终日昏昏醉梦间，忽闻春尽强登山。
因过竹院逢僧语，又得浮生半日闲。
```

### 鱼鼓颂｜从谂

```
四大由来造化工，有声全贵里头空。
莫嫌不与凡夫说，只为宫商调不同。
```

### 将赴吴兴登乐游原一绝｜杜牧

```
清时有味是无能，闲爱孤雲静爱僧。
欲把一麾江海去，乐游原上望昭陵。
```

### 尘劳迥却｜希运

```
尘劳迥却事非常，紧把绳头做一场。
不是一番寒彻骨，争得梅花扑鼻香。
```

### 空门不肯出｜神赞

```
空门不肯出，投窗也大痴。
百年钻故纸，何日出头时。
```

### 过水偈｜良价

```
切忌从他觅，迢迢与我疏。
我今独自往，处处得逢渠。
渠今正是我，我今不是渠。
应须凭么会，方得契如如。
```

### 滔滔不持戒｜慧寂

```
滔滔不持戒，兀兀不坐禅。
酽茶三两碗，意在镢头边。
```

### 松柏操｜慧忠

```
雪後始知松桂别，雲收方见济河分。
不因世主教还俗，哪辨鸡群与鹤群!
```

### 送僧清演归山｜李洞

```
毛褐斜肩背负经，晓思吟入窦山青。
峰前野水横官道，踏着秋天三四星。
```

### 焰里寒冰结｜本寂

```
焰里寒冰结，杨花九月飞。
泥牛吼水面，木马逐风嘶。
```

### 家在闽山｜怀浚

```
家在闽山东复东，其中岁岁有花红。
而今再到花红处，花在旧时红处红。
```

### 醉着｜韩偓

```
万里清江万里天，一村桑柘一村烟。
渔翁醉着无人唤，过午醒来雪满船。
```

### 闻子规｜杜荀鹤

```
楚天空阔月沉轮，蜀魄声声似告人。
啼得血流无用处，不如缄口过残春。
```

### 慈恩寺偶题｜郑谷

```
往事悠悠添浩叹，劳生扰扰竟何能。
故山岁晚不归去，高塔晴来独自登。
林下呼经秋苑鹿，江边扫叶夕阳僧。
吟余却起双峰念，曾看庵西瀑布冰。
```

### 片雲｜齐己

```
水底分明天上雲，可怜形影似吾身。
何妨舒作从龙势，一雨吹销万里尘。
```

### 金屑眼中翳｜文偃

```
金屑眼中翳，衣珠法上尘，
己灵犹不重，佛祖为何人？
```

### 示僧偈｜景岑

```
百尺竿头坐底人，虽然得入未为真。
百尺竿头须进步，十方世界现全身。
```

### 示徒颂｜神禄

```
萧然独处意沉吟，谁信无弦发妙音。
终日法堂唯静坐，更无人间本来心。
```

### 清风楼上｜师鼐

```
清风楼上赴官斋，此日平生眼豁开。
方信普通年事远，不从葱岭带将来。
```

### 拥毳对芳丛｜文益

```
拥毳对芳丛，由来趣不同。
发从今日白，花是去年红。
艳冶随朝露，馨香逐晚风。
何须待零落，然後始知空。
```

### 开悟诗｜清海

```
实际从来不受尘，个中无旧亦无新。
青山况是吾家物，不用寻家别问津。
```

### 闲居｜延寿

```
闲居谁似我，退迹理难过。
要势危身早，浮荣败德多。
雨催虫出穴，寒逼鸟移窠。
野径无人翦，疏窗入薜萝。
```

### 居天柱山｜赞宁

```
四野豁家庭，柴门夜不扃。
水边成半偈，月下了残经。
虽逐诸尘转，终归一念醒。
未知斯旨者，万役尽劳形。
```

### 送僧｜士可

```
一钵即生涯，随缘度岁华。
是山皆有寺，何处不为家。
笠重吴天雪，鞋香楚地花。
他年访禅师，宁惮路歧赊。
```

### 岁暮自桐庐归钱塘｜潘阆

```
久客见华发，孤棹桐庐归。
新月无朗照，落日有余晖。
渔浦风水急，龙山烟火微。
时闻沙上雁，一一背人飞。
```

### 宿西山精舍｜文兆

```
西山乘兴宿，静兴寂寥心。
一径松杉老，三更雨雪深。
草堂僧语息，雲阁磬声沉。
未遂长栖此，双峰晓待寻。
```

### 酬赠梦真上人｜行肇

```
禅舍因吟往，晴来坐彻宵。
春通三径晚，家别九江遥。
巢重禽初宿，窗明叶旋飘。
佳期应未定，谢守有诗招。
```
